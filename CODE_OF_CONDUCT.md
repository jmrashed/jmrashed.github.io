# Code of Conduct

## Our Pledge

We pledge to make participation in our project a harassment-free experience for everyone.

## Our Standards

- Be respectful and inclusive
- Communicate professionally
- Accept constructive criticism gracefully
- Focus on what is best for the community

## Our Responsibilities

Project maintainers are responsible for clarifying standards and enforcing them fairly.

## Scope

This Code of Conduct applies within all project spaces.

## Enforcement

Instances of abusive or harassing behavior should be reported to the project maintainers.
